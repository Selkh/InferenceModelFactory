# AFW 数据集准备
## 步骤

1. 下载 [AFW](https://drive.google.com/open?id=1Kl2Cjy8IwrkYDwMbe_9DVuAwTHJ8fjev)
2. 解压afw_images.zip文件 `unzip -d <data/AFW> <afw_images.zip>`, `mv data/AFW/afw_images data/AFW/images`，目录结构如下：
    ```shell
    data
        └── AFW
            └── images
                ├── 1004109301.jpg
                ├── 1051618982.jpg
                ├── ...
                ├── README
                └── anno.mat
    ```
